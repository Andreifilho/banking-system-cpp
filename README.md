# Parking Lot Management System (C++)

This project is a console-based C++ program that simulates a parking lot using a 2D array. It tracks occupied and free parking spots and displays available locations.

## Features
- 2D array representation of a parking lot
- Tracks occupied and free spaces
- Displays total, occupied, and free spot counts
- Lists available parking locations

## How to Run
Compile and run using a C++ compiler:

g++ parking_lot.cpp -o parking_lot
./parking_lot
